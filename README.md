# PixelNeighbor

This project is a simple web application that displays a "Hello World" message. It is designed to be hosted on GitHub Pages and is connected to the custom domain pixelneighbor.com.

## Project Structure

- **index.html**: The main HTML file that contains the structure of the web page.
- **CNAME**: A file that specifies the custom domain for the GitHub Pages site.
- **css/styles.css**: Contains styles for the web page.
- **js/main.js**: Contains JavaScript code for interactivity.
- **.github/workflows/github-pages.yml**: GitHub Actions workflow for automating deployment to GitHub Pages.
- **README.md**: Documentation for the project.
- **LICENSE**: Licensing information for the project.

## Getting Started

To get started with this project, clone the repository and open the `index.html` file in your web browser. You can also customize the styles in `css/styles.css` and add interactivity using `js/main.js`.

## Deployment

This project is set up to be deployed on GitHub Pages. Ensure that the `CNAME` file contains your custom domain (pixelneighbor.com) for proper configuration.
